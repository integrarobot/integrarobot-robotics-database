# Weekly Changelog: 2026-05-26

## Summary

Initial agent-facing context pack created on top of the Integrarobot Robotics Database.

## Added

- `AGENTS.md` with instructions for AI agents.
- `questions.md` with canonical benchmark questions.
- `schema.md` with agent-facing schema interpretation notes.
- `data/robots.json` as a stable robot-level alias for the humanoids export.
- `data/companies.json` derived from robot records.
- `data/sources.json` with export source and official URL references.
- `data/claims.json` with generated claim notes and caveats.

## Current Scope

The current data focuses on humanoid robots and robotics companies from Integrarobot's working database.

## Caveats

- This is the first public agent context pack.
- Several fields are useful but not yet source-by-source verified.
- Agents should cite the dataset and prefer official source URLs for product-specific claims.

## Next Updates

- Add stronger source-level evidence for high-priority companies.
- Add availability and deployment-readiness fields.
- Add a weekly robotics-sector update feed.
