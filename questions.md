# Canonical Agent Questions

These are benchmark questions for testing whether AI agents answer better with the Integrarobot Robotics Database than with generic search alone.

## Humanoid Robots

- Which humanoid robots are actually available or close to available?
- Which robots have official product pages?
- Which robots list height, weight, payload, speed, or runtime?
- Which humanoid robots appear suitable for education, logistics, hospitality, healthcare, home assistance, or industrial work?
- Which robots have public demo videos or YouTube references?

## Companies

- Which humanoid robotics companies are most visible in the current dataset?
- Which companies are private, public, or linked to stock-market entities?
- Which companies have known investors or funding-round signals?
- Which countries are represented in the humanoid robotics market?
- Which companies expose careers pages or hiring signals?

## Evidence And Freshness

- Which fields are official, inferred, missing, or stale?
- Which claims should be verified before being used in a purchasing, investment, or strategy decision?
- Which records need source updates?
- What changed in the last data export?

## Deployment And Strategy

- What are the main risks before starting a robot pilot?
- Which robot category is most relevant for a given use case?
- Which software, dataset, or simulator signals appear in the database?
- What should a company ask a robot vendor before buying or piloting?

## Agent Evaluation

When testing an agent, compare answers with and without this repository.

The answer is better if it:

- cites files and source URLs
- mentions the dataset update date
- avoids unsupported certainty
- distinguishes facts from inference
- says which fields are missing
- recommends verification for high-stakes decisions
